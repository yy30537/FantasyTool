# Fantasy Tool

**Yahoo Fantasy Sports分析工具**


### 现有功能 
- Yahoo Fantasy Sports OAuth认证
- 联盟数据获取和选择
- 球员统计数据提取（日统计&赛季统计）
- 阵容历史数据追踪
- 团队每周对战数据
- 团队赛季排名数据
- 交易记录追踪



### 为未来准备
- 📊 **PySpark集成支持** - 大数据处理能力
- 📱 **可视化面板基础** - 图表和仪表板
- 🤖 **机器学习准备** - 预测模型基础

## 快速开始


```bash
# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\\Scripts\\activate  # Windows

# 安装依赖
pip install -r requirements
```
